Esse é o meu primeiro arquivo completo em Python, o projeto é de uma calculadora simples que contém soma, subtração, multiplicação e divisão, e logo em seguida quando a pessoa não quiser usar
mais a calculadora, há uma pergunta para caso a pessoa quiser usar também o conversor de temperatura, de Celsius para Fahrenheit e de Fahrenheit para Celsius. Tentei prevenir todos os bugs
possíveis e deixar a documentação bem clara, cada entrada tem seu tipo, e as que não tem, há funções que servem como previnidores de erro, que se a resposta não for a esperada, da loop no
programa pedindo para o usuário tentar de novo expecificando novamente qual tipo de resposta esperada.
